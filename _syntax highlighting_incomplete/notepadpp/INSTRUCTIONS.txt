
To install:
1. Open any folder and write in the direction bar %appdata%/notepad++/userDefineLangs
2. put the userDefineLang.xml inside this folder

Open notepad++
Go to settings->preferences.
Click the backup/autocompletion tab.
Under auto-completion enable the following checkboxes:
-Enable autocomplete on each input
-Function parameters hint on input
You're done!
